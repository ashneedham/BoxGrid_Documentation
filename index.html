a flexbox grid for rapid website development

how to use

BoxGrid is really simple to install as it consists of 1 core css file and if you want to support IE9 then 1 extra css + 1 js file + jQuery.

<link href="css/boxgrid.min.css" rel="stylesheet" type="text/css">
<!--[if IE 9]> <link href="css/boxgrid-ie.min.css" rel="stylesheet" type="text/css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
<script type="text/javascript" src="js/boxgrid-ie-fix.min.js"></script> <![endif]-->

Responsive Break-points

The grid uses 5 break-points to allow for varying layouts at different screen sizes. Apply the prefix followed by the number of column span required, to each column.

Styles for each break-point override the previous.

Name Class prefix Screen size
X-Small xs_ >= 300px
Small sm_ >= 450px
Medium md_ >= 750px
Large lg_ >= 1200px
X-Large xl_ >= 1600px

Column sizes

All columns must have a class of col. By default a column will be 100% wide.

Apply a prefixed number to specify how many columns a particular column will span e.g. md_4 (medium span 4).

Column sizes range from 0 (zero) to 12 (twelve).

Pre column spacing

Add blank spaces to the left of a column using the pre-spacing values.

To use pre-spacing, add a class constructed of the break-point prefix along with a prefix of pre_ followed by the number of columns spacing required (0 - 11).

9
7
4
3
<div class="boxgrid"> <div class="col xs_9 xs_pre_3">9</div>
<div class="col xs_7 xs_pre_5">7</div>
<div class="col xs_4 xs_pre_8">4</div>
<div class="col xs_3 xs_pre_9">3</div>
 </div>

Post column spacing

You may wish to prevent any further spacing, after a column, from being used. This is particularly useful if you wish to create the effect of a new row.

To apply post-spacing, add a class constructed of the break-point prefix along with a prefix of post_ followed by the number of columns space required (0 - 11).

8
4
3
<div class="boxgrid"> <div class="col xs_8 xs_post_4">8</div>
<div class="col xs_4 xs_post_8">4</div>
<div class="col xs_3">3</div>
 </div>

Changing column order

You can change the order of columns using data attributes, keeping the HTML in the order you want but simply swapping the columns around using styles.

Add a data attribute of data-bg_xs_order="2" to the HTML element (replacing xs_ with the relevant break-point prefix and the 2 with the order number required). You can use an order number of 1 through to 12.

One
Two
Three
Four
<div class="boxgrid"> <div class="col xs_3" data-bg_xs_order="4" data-bg_sm_order="2" data-bg_md_order="3" data-bg_lg_order="4" data-bg_xl_order="4">One</div>
<div class="col xs_3" data-bg_xs_order="3" data-bg_sm_order="3" data-bg_md_order="4" data-bg_lg_order="1" data-bg_xl_order="2">Two</div>
<div class="col xs_3" data-bg_xs_order="2" data-bg_sm_order="4" data-bg_md_order="1" data-bg_lg_order="2" data-bg_xl_order="3">Three</div>
<div class="col xs_3" data-bg_xs_order="1" data-bg_sm_order="1" data-bg_md_order="2" data-bg_lg_order="3" data-bg_xl_order="1">Four</div> </div>

Alternatively you can apply a class of first or last (again prefixed with a break-point prefix) to move the column to the start or the end. Please bear in mind that this will take into account all columns inside one particular grid, not just on a row.

One
First
Last
Two
<div class="boxgrid"> <div class="col xs_3">One</div>
<div class="col xs_3 xs_first">First</div>
<div class="col xs_3 xs_last">Last</div>
<div class="col xs_3">Two</div> </div>

Content alignment

You can change the horizontal and vertical alignment of content within a column, and this can differ per breakpoint.

It is recommend to wrap the content of an aligned column in a containing block element. This is because an aligned column applies flex which causes it's content to stack in a row.

Horizontal alignment

To apply horiztonal alignment apply a class built up of break-point prefix followed by left, center or right.

Left aligned
content
Centre aligned
content
Right aligned
content
<div class="boxgrid"> <div class="col xs_12 xs_left">Left aligned<br>content</div>
<div class="col xs_12 xs_center">Centre aligned<br>content</div>
<div class="col xs_12 xs_right">Right aligned<br>content</div> </div>

Vertical alignment

To apply vertical alignment apply a class built up of break-point prefix followed by top, middle or bottom.

Top
Middle
Bottom
<div class="boxgrid"> <div class="col xs_4 xs_top" style="height: 170px;">Top</div>
<div class="col xs_4 xs_middle">Middle</div>
<div class="col xs_4 xs_bottom">Bottom</div> </div>
